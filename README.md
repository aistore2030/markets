# markets
Related to crypto and bank currency markets.

These scripts use the latest version of Bash
and show a help page if run with "-h".


bcalc.sh -- Simple Bash calculator taht uses bc and keeps a record of results;

binance.sh -- Access Binance public API, prices, book depth, coin ticker;

binfo.sh -- Access Blockchain.info (Blockchain.com) public API;

bitstamp.sh -- Access Bitstamp live price roll API;

cgk.sh -- Bash currency converter using CoingGecko.com public API, market ticker;

clay.sh -- Bash central bank currency, precious metal and (BTC) crypto converter using CurrencyLayer.com API;

cmc.sh -- Bash currency converter using CoinMarketCap.com API, market ticker;

erates.sh -- Bash currency converter using ExchangeRates.com public API;

myc.sh -- Bash central bank currency converter using MyCurrency.net public API;

openx.sh -- Bash central bank currency converter using MyCurrency.net public API
